# greenct-testcases
A beginner-friendly collection of test cases for practicing software testing concepts and improving QA skills. This repository includes simple, well-structured examples designed to help students understand testing basics, write clear test scenarios, and build confidence in quality assurance practices.
Короткий опис: Створити три тест-кейси для перевірки функціональності системи. Кожен тест-кейс має містити опис передумов, кроків виконання та очікуваного результату. Тест-кейси повинні охоплювати різні варіанти поведінки системи, відображаючи типові та можливі відхилення у її роботі.
Посилання на сторінку, що тестується: https://www.greencity.cx.ua/#/greenCity/events

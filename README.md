# greenct-testcases
Короткий опис: Створити три тест-кейси для перевірки функціональності системи, а також автотести для їх прикладної реалізації. Кожен тест-кейс має містити опис передумов, кроків виконання та очікуваного результату. Тест-кейси повинні охоплювати різні варіанти поведінки системи, відображаючи типові та можливі відхилення у її роботі.
Посилання на сторінку, що тестується: https://www.greencity.cx.ua/#/greenCity/events

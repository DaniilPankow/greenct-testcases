# Test-case3: Створення івенту

## Preconditions
- Користувач авторизований у системі
- Відкрита сторінка "Events"
- Існує можливість створення івенту

## Test Steps

| Step | Action                                 | Data            | Expected Result                                   |
|------|----------------------------------------|-----------------|---------------------------------------------------|
| 1    | Відкрити сторінку                      | URL             | Відображається сторінка сайту                     |
| 2    | Натиснути на пункт меню "Events"       | URL             | Відображається увесь список івентів               |
| 3    | Натиснути на кнопку "Create Event"     | URL             | Відображається форма створення івенту             |
| 4    | Заповнити поле Title                   | Валідні дані    | Дані введені в поле                               |
| 5    | Вибрати термін тривалості              | Список значень  | Термін тривання обрано з переліку                 |
| 6    | Обрати тематику                        | Список значень  | Обрано доступну тематику з переліку               |
| 7    | Обрати доступність івенту              | Список значень  | Варіант доступності вибрано з переліку            |
| 8    | Написати опис івенту                   | Текст           | Написано та відображено текст                     |
| 9    | Обрати або завантажити картинку        | IMG/PNG         | Добавлено або вибрано картинку                    |
| 10   | Ввести дату початку та кінця           | Calendar        | Дата початку та кінця установлено                 |
| 11   | Вибрати формат                         | Список значень  | Відповідний формат обрано                         |
| 12   | Ввести додаткову інформацію для формату| Валідні дані    | Дані для формату введено                          |
| 13   | Натиснути на кнопку "Preview"          | Click           | Попередній огляд івенту                           |
| 14   | Натиснути на кнопку "Publish"          | Click           | Івент успішно додано до пункту "Events"           |
